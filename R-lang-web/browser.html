9.1 浏览

在函数 browser 调用的地方，R 会暂停执行， 并且为用户提供特别的提示。赋给 browser 的参数会被自动 忽略。 

     > foo <- function(s) {
     + c <- 3
     + browser()
     + }
     > foo(4)
     Called from: foo(4)
     Browse[1]> s
     [1] 4
     Browse[1]> get("c")
     [1] 3
     Browse[1]>
