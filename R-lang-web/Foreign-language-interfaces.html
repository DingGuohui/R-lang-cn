7.2 其它语言接口

用编译后的代码为R增加函数的 技术细节可以参考 System and foreign language interfaces (Writing R Extensions)。

函数 .C 和 .Fortran 提供了标准接口，允许 在程序构建时（build time）使用外部代码或者通过dyn.load连接 R 和 已编译的代码。这个两个函数的设计最初是想对 C 和 FORTRAN 的代码分别编译，但是 .C 函数可用于其它能产生 C 接口的语言中， 如 C++。

函数 .Call 和 .External 提供了在允许已编译代码 （主要是编译后的C代码）中 操作R对象的接口。
