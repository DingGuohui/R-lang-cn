10.4.5 流控制元素

R 含有作为特殊语法结构的流控制结构 

     if ( cond ) expr
     if ( cond ) expr1 else expr2
     while ( cond ) expr
     repeat expr
     for ( var in list ) expr

这些结构中的表达式是典型的复合表达式。

在循环结构（while，repeat，for）中， 我们还可以使用 break （终止循环） 和 next（忽略本次循环进入下次循环）。

在内部，这些结构都以函数调用的方式保存： 

     "if"(cond, expr)
     "if"(cond, expr1, expr2)
     "while"(cond, expr)
     "repeat"(expr)
     "for"(var, list, expr)
     "break"()
     "next"()
