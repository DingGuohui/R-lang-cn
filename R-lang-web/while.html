3.2.4 while语句

while 语句和 repeat 语句非常的类似。 while 循环的语法如下 

     while ( statement1 ) statement2

其中 statement1 会被求值，如果它的值是 TRUE那么 statement2 会被执行。这个过程重复直到 statement1 的值是 FALSE。如果 statement2 从来都没有被执行， 那么 while语句返回NULL，否则它将会返回 最后一次执行 statement2 所得到的值。
